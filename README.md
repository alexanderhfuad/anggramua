# Anggra MUA Monorepo

Landing page modern dan dinamis untuk brand **Anggra MUA** dengan arsitektur monorepo:

- `frontend`: React + Tailwind + Vite
- `backend`: Express API sederhana

## Fitur

- SEO-friendly meta tags (title, description, Open Graph, canonical)
- Dark/Light mode toggle
- Dual language: Bahasa Indonesia & English
- Floating WhatsApp icon + CTA booking
- Desain modern dengan efek glassmorphism dan gradient
- Backend API untuk data layanan (`/api/services`)

## Menjalankan Project

1. Install dependency:

```bash
npm install
```

2. Jalankan frontend + backend bersamaan:

```bash
npm run dev
```

3. URL default:

- Frontend: `http://localhost:5173`
- Backend: `http://localhost:4000`
